# asignment3
Основна частина у файлі `solution.sql` та включає:
1. **Функції:** `calculate_order_total` для динамічного обчислення загальної вартості замовлення на основі товарів у кошику.
2. **Збережені процедури:** * `create_order`: Безпечно створює нове замовлення, попередньо перевіряючи, чи існує такий клієнт.
   * `add_product_to_order`: Додає товари до замовлення, автоматично списуючи необхідну кількість зі складу (`stock_quantity`). Використовує CTE (конструкцію `WITH`) для ефективного виконання без використання змінних.
3. **Тригери:**
   * `trg`: Автоматично перераховує `total_amount` у таблиці `orders` щоразу, коли товари додаються, оновлюються або видаляються з `order_items`.
   * `trg2`: Тригер аудиту, який автоматично створює запис у таблиці `order_log` при кожному успішному створенні нового замовлення.
